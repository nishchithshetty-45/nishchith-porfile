# Nishchith U Shetty — GTM Portfolio

A single-page portfolio site for Nishchith U Shetty, GTM & Automation Specialist.

## 📁 Structure
```
/
├── index.html        ← the whole site (no build step needed)
├── images/
│   └── profile.jpg   ← ADD YOUR PHOTO HERE before deploying
└── README.md
```

## ⚠️ Before you push to GitHub
1. Drop your photo into `images/profile.jpg` (must be named exactly that, or update the `src` in `index.html`)
2. Update the LinkedIn URL and email if they change
3. (Optional) Replace the favicon / OG image if you want something custom

## 🚀 Deploy to Vercel
1. Push this folder to a GitHub repo
2. Go to vercel.com → **Add New Project** → import your GitHub repo
3. Framework preset: **Other** (it's static HTML, no build command needed)
4. Click **Deploy**

That's it — no `npm install`, no build step, no env vars. Vercel will serve `index.html` directly.

## 🛠 Local preview
Just open `index.html` in a browser, or run:
```bash
npx serve .
```
